# Psychology & AI System (Diplom Project)

Этот проект представляет собой систему для поддержки психического здоровья с использованием искусственного интеллекта.
Проект состоит из бэкенда на FastAPI и фронтенда на React (Vite).

## Структура проекта

- `backend/` — Серверная часть на FastAPI.
- `Frontend/` — Клиентская часть на React.

## Требования

Для запуска проекта вам понадобятся:

- **Python 3.10+**
- **Node.js** (рекомендуется v18 или выше)
- **PostgreSQL** (убедитесь, что сервер запущен)

---

## Инструкция по запуску

### 1. Подготовка Базы Данных

Создайте пустую базу данных в PostgreSQL (например, с названием `mental_health`).

### 2. Настройка Бэкенда

Перейдите в директорию бэкенда:

```bash
cd backend
```

Создайте виртуальное окружение и активируйте его:

```bash
# Для macOS/Linux:
python3 -m venv venv
source venv/bin/activate

# Для Windows:
python -m venv venv
venv\Scripts\activate
```

Установите необходимые зависимости:

```bash
pip install -r requirements.txt
```

Настройте переменные окружения:

1. Создайте файл `.env` в папке `backend/` на основе `.env_EXAMPLE`.
2. Заполните параметры:
    - `DATABASE_URL`: `postgresql://username:password@localhost:5432/mental_health`
    - `SECRET_KEY`: Любая длинная случайная строка для защиты токенов.
    - `OPENAI_API_KEY`: Ваш ключ API от OpenAI.

Запустите сервер:

```bash
uvicorn src.main:app --reload
```

* Бэкенд запустится на: `http://localhost:8000`
* Swagger UI (документация): `http://localhost:8000/docs`

### 3. Настройка Фронтенда

Перейдите в директорию фронтенда:

```bash
cd Frontend
```

Установите зависимости:

```bash
npm install
```

Запустите приложение:

```bash
npm run dev
```

* Приложение будет доступно по адресу: `http://localhost:5173`

---

## Основные возможности

- **Дневник настроения**: Отслеживание эмоционального состояния.
- **ИИ-Ассистент**: Поддержка и советы на базе GPT.
- **Личный кабинет**: История записей и аналитика.
- **Панель специалиста**: Работа терапевтов с данными пациентов.
